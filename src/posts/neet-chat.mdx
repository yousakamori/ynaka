## 1.サービスの概要

**Firebase で作成したチャット**です。

- [サイト URL](https://neetschat.firebaseapp.com)
- [Github](https://github.com/yousakamori/vue-firebase-chat)

Firebase の勉強のため数年前に遊びで作ったものです。

## 2.使用技術

### フロントエンド

- Vue.js 2.6.10
- Vuetify 1.5.5
- Vuex 3.0.1

### インフラ

- Firebase
  - Firestore
  - Authentication
  - Hosting

### その他

- Git / GitHub
- VSCode
